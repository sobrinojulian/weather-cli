# Weather-cli

> Check the local weather from your terminal

## Usage

```
$ weather --help

  Usage
    $ weather [command]...

  Commands
    -c, --current
    -f, --forecast=<1|2|3|4>

  Examples
    $ weather
    ...
    $ weather -c
    ...
    $ muerte -f 1 2 4
    ...
```
